# Developer Portofolio & Admin Engine

## Tech Stack & Infrastructure
### Orchestration
* **Docker Compose:** Containerized local env

### Database (PostgreSQL)
* **Migrations:** Enterprise-grade migration system utilizing `golang-migrate`.
* **Advanced Schema:** Features UUIDs, `JSONB` for robust metrics and architecture data, and `TEXT[]` arrays designed to perfectly capture the *Problem → Role → Key Decision → Outcome* narrative of a senior engineer.

### Backend (Go 1.26)
* **REST API:** Blazing-fast endpoints built using Go's native routing.
* **Development Workflow:** Configured with `air` for instantaneous hot-reloading within Docker.
* **Data Handling:** Successfully bridges complex strict-typing challenges, mapping JavaScript arrays to Go slices and securely parsing JSON payloads.

### Frontend (React + Vite)
* **Architecture:** Multi-page Admin CMS engineered with `react-router-dom`.
* **Styling & UI:** Tailwind CSS integrated and polished components.
